MyFolder - 轻量级私有云存储系统
MyFolder 是一个基于 Web 的简易文件管理系统，旨在提供安全、高效的本地化文件存储与管理服务。它允许用户在私有环境中轻松地对文件资源进行操作，并提供了完善的权限验证机制。

✨ 核心功能 (Features)
目前版本已实现核心的文件系统操作与安全控制：

🗂 文件管理：支持文件与文件夹的上传 (Upload)、下载 (Download) 及删除 (Delete)。

📋 剪贴板操作：实现了类似操作系统的文件/文件夹复制 (Copy) 与 粘贴 (Paste) 逻辑。

📂 目录结构：支持在线创建新文件夹，灵活管理目录层级。

🔒 权限控制：内置身份验证机制，访客仅读，登录用户拥有写入权限（对目标文件夹的增删改操作均需鉴权）。

🚀 路线图 (Roadmap)
我们正在积极开发 v1.0.0 正式版，主要聚焦于降低部署门槛与提升用户体验：

[ ] 前后端一体化：将 Vue 前端资源与 Spring Boot 后端服务进行深度整合，移除对 Nginx/Node.js 环境的依赖。

[ ] 一键部署 (One-Click Deployment)：

封装嵌入式运行环境（Embedded JRE, MySQL, Redis）。

发布适用于 Windows (.exe) 和 Linux (.sh) 的离线安装包，实现“开箱即用”的零配置部署体验。
